# Gestion de fichier

**Durée :** 4 heures  
**Travail en groupe :** 2 personnes  

---

## Technologie utilisée
- Python  

---

## Objectif du projet

Transformer un fichier **JSON** en un fichier **CSV** contenant les colonnes suivantes :

- PathologieLaPlusPrécise  
- NomComorbidité  
- Année  
- EffectifPathologie  
- EffectifComorbidité  
- Proportion  
- niveauPrioritaire  

---

## Description du projet

Pour ce projet, nous disposions d’un fichier **JSON** contenant un jeu de données de patients et leurs pathologies.

Nous devions :
- ouvrir le fichier en **Python**
- nettoyer et trier les données
- ne conserver que les lignes contenant l’ensemble des informations demandées

Pour cela, nous avons utilisé :
- des **listes de listes**
- des **boucles**
- des **conditions**

Chaque ligne était traitée individuellement afin de vérifier la présence de toutes les informations nécessaires.

Une fois le traitement terminé, nous avons exporté la liste finale sous la forme d’un **fichier CSV**.
