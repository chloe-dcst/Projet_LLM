# Conception et implémentation d’une Base de Données Relationnelle (BDR)

**Durée :** 4 semaines  
**Travail en groupe :** 3 personnes  

---

## Technologies utilisées
- Python  
- MySQL  
- SQL  
- Word  

---

## Objectif du projet

Améliorer les moyens de transmission de l’alerte du **SDIS79**.  
Pour cela, nous devions :
- créer une **base de données relationnelle**
- développer une **application (interface)** permettant d’échanger avec cette base de données

---

## Description du projet

Dans le cadre de ce projet, nous avons d’abord élaboré un **modèle conceptuel de données** à l’aide du logiciel **Looping**, ce qui nous a permis de concevoir la structure de la base de données.

À partir de ce modèle, nous avons généré le **modèle relationnel en SQL**, puis l’avons importé dans **MySQL** afin de créer la base de données *SDIS79*.

Nous avons ensuite :
- créé un **jeu de données** inséré en SQL
- utilisé **Python** pour importer et traiter un autre jeu de données fourni sous forme de fichiers CSV
- nettoyé et modifié ces fichiers afin qu’ils soient compatibles avec la base de données

Une fois la base de données complétée, nous avons rédigé **10 requêtes SQL** permettant d’extraire des informations pertinentes.

Nous avons également conçu une **interface utilisateur** permettant :
- de visualiser le contenu d’une table
- d’insérer un nouveau tuple dans une table

Enfin, nous avons rédigé un **rapport détaillant l’ensemble du projet**.
