# Création d'une application web sur les AirBnb

**Tags :** HTML/CSS, JavaScript/Plotly  
**Durée :** 2 semaines | **Groupe :** 4 personnes

## L'objectif
Créer une application web sur les statistiques des logements AirBnb à Paris en utilisant la technologie Plotly.

### La description du projet
Pour réaliser cette application web, nous devions utiliser le framework Plotly de JavaScript et respecter quelques contraintes :  

- **Premier onglet** : ensemble de graphiques et tableaux liés sur les variables suivantes avec filtres : type de logement (`room_type`), prix (`price`), et d’autres informations pertinentes pour un décideur AirBnb, avec si possible une carte globale de Paris indiquant chaque logement.  
- **Deuxième onglet** : carte choroplèthe, où l’on peut choisir la statistique représentée (nombre de logements, prix moyen, part de logement entier).  
- **Troisième onglet** : représentation des parts de type de logement par arrondissement sous forme de "small multiples".  

Nous avons choisi de créer une page différente pour chaque onglet. Tout fonctionne correctement, mais pour des améliorations futures, nous aurions voulu que, sur la page Analyse des données, lorsque l’on choisit un type de logement, certains graphiques camemberts disparaissent (au moins celui de gauche) car ils deviennent moins pertinents.
