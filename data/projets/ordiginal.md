# Création d'un site web dynamique

**Tags :** HTML/CSS, PHP, JavaScript, MySQL  
**Durée :** 7 semaines | **Groupe :** 3 personnes

## L'objectif
Concevoir un site web complètement dynamique sur un sujet au choix à partir d'un jeu de données issu de l'OpenData.  
Il fallait réaliser le diagramme UML, créer une base de données relationnelle avec MySQL, créer un site web en HTML/CSS, utiliser PHP pour les formulaires et JavaScript pour réaliser un tableau de bord.

### La description du projet
Nous avons choisi de créer une application web sur les ordinateurs pour conseiller une personne dans le choix d'un ordinateur.  
Elle permet également de voir l'avis des clients ayant acheté un ordinateur et de visualiser les modèles les plus populaires grâce à une page statistique.  

Comme nous n'avions pas de base OpenData, nous avons conçu notre propre base comprenant 4 tables : `ordi`, `client`, `commande` et `avis`.  
Chaque table a été insérée dans MySQL via easyphpMyAdmin.  

Pour le site web :  
- **HTML/CSS** : mise en page et design  
- **PHP** : formulaires et gestion des utilisateurs  
- **JavaScript/jQuery** : graphiques du tableau de bord  

Le site est structuré en deux parties :  
- **Utilisateur** : pages "Accueil", "Recherche" (formulaire de sélection d’ordinateurs), "Commandes", "Avis" et "Statistiques" (graphiques jQuery)  
- **Ordiginal Pro** : gestion des tables `client`, `ordi` et `commande`, avec ajout, modification et suppression d'éléments  

Enfin, nous avons rédigé un rapport détaillant toutes les étapes, le diagramme UML, la répartition du travail, les fonctionnalités et les améliorations possibles.
